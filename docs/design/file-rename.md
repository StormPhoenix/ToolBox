# file-rename 插件设计文档

## 1. 功能概述

`file-rename` 是一个批量文件重命名工具，支持四种重命名规则，并提供实时预览，只有用户点击「执行重命名」后才会真正写入磁盘。

---

## 2. 功能规格

### 2.1 文件导入

| 方式 | 说明 |
|------|------|
| 点击「添加文件」按钮 | 通过 `showOpenDialog` 多选文件或选文件夹 |
| 拖拽到界面 | 支持拖入文件和文件夹 |

- 若选择/拖入的是**文件夹**，只列出其第一层文件，不递归子目录
- 新导入的文件按**文件名字典序**排序后追加到列表末尾，同路径文件去重

### 2.2 重命名规则（四选一）

每次只能激活一种规则，规则切换时配置重置。

| 规则 | 配置项 | 行为说明 |
|------|--------|----------|
| **查找替换** | 查找字符串、替换字符串 | 大小写敏感，普通字符串全部替换；留空替换字符串则删除该字符串 |
| **前后缀** | 前缀、后缀 | 在文件名（不含扩展名）前后分别追加字符串 |
| **序号模式** | 起始数字、是否补零 | 将整个文件名（不含扩展名）替换为序号；开启补零则自动根据最大数字位数补零 |
| **手动编辑** | — | 在预览列表中直接点击新名称列进行输入编辑 |

**规则叠加**：手动编辑可与其他三种规则叠加——其他规则先生成基础新名，用户在此基础上手动微调某几项（覆盖该 item 的规则结果）。

### 2.3 预览区

- 每行显示：拖拽把手 / 文件图标 / 原文件名 → 新文件名 / 移除按钮
- 新文件名实时响应规则配置变化
- 存在重名冲突时，冲突行高亮红色边框，底栏同步提示，「执行重命名」按钮禁用
- 支持 SortableJS 拖拽排序（影响序号分配顺序）

### 2.4 执行重命名

- 用户点击「✏️ 执行重命名」按钮触发
- 遍历列表，对名称有变化的文件调用 `fs:renameFile` IPC
- 成功后清空列表，Toast 提示「重命名完成」
- 部分失败时 Toast 提示错误，列表保留未成功的文件
- **不支持撤销**

## 3. 交互设计

### 3.1 整体布局（上下分区）

```
┌──────────────────────────────────────────────────┐
│ 顶栏（52px）：标题 / 文件数 / 添加文件 / 清空     │
├──────────────────────────────────────────────────┤
│ 规则区（固定）：规则 Tab + 当前规则配置表单       │
├──────────────────────────────────────────────────┤
│ 预览区（flex:1 可滚动）：文件卡片列表             │
├──────────────────────────────────────────────────┤
│ 底栏（固定）：冲突提示 / 执行重命名按钮           │
└──────────────────────────────────────────────────┘
```

### 3.2 状态流转

```
空状态（drop-zone）
  ↓ 导入文件
有文件状态（规则区 + 预览区 + 底栏）
  ↓ 点击「执行重命名」
执行中（按钮禁用 + loading 文字）
  ↓ 全部成功
空状态（列表清空）+ 成功 Toast
  ↓ 部分失败
保留失败项 + 错误 Toast
```

## 4. 技术方案

### 4.1 文件结构

```
plugins/builtin/file-rename/
├── manifest.json
├── package.json
├── tsconfig.json
├── vite.config.ts
└── src/
    ├── index.html
    ├── main.ts
    ├── App.vue
    └── composables/
        ├── useFileItems.ts       # 文件列表状态、导入、规则计算、预览名
        └── useRenameExecute.ts   # 执行重命名逻辑
```

### 4.2 新增 IPC

| 方法 | IPC 通道 | 说明 |
|------|----------|------|
| `renameFile(oldPath, newPath)` | `fs:renameFile` | 主进程调用 `fs.rename` 重命名文件 |

修改点：`src/main/main.ts`、`src/main/preload.ts`、`plugins/shared/bridge/src/types.ts`

### 4.3 依赖

| 包 | 用途 |
|---|---|
| `sortablejs` | 预览列表拖拽排序 |
| `@toolbox/bridge` | IPC 通信 |
