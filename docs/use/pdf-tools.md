# PDF 工具使用说明

ToolBox 内置两个独立的 PDF 工具插件，分别承担不同职责：

| 插件 | ID | 功能 |
|------|----|------|
| PDF 编辑 | `pdf-editor` | 对单个 PDF 进行页面删除、插入、重排后导出 |
| PDF 拆分 | `pdf-split` | 将单个 PDF 按规则拆分为多个独立文件 |

---

## PDF 编辑（pdf-editor）

### 打开文件

- 点击工具栏 **📂 打开文件** 按钮，通过文件对话框选择 PDF
- 或将 PDF 文件直接**拖拽**到工具区域

### 页面操作

在左侧缩略图面板上右键单击任意页面，弹出上下文菜单，可执行：

| 操作 | 说明 |
|------|------|
| 移除页面 | 将该页标记为删除（显示红色遮罩），保存时不输出 |
| 插入空白页 | 在当前页前/后插入一张空白 A4 页 |
| 插入图片页 | 选择 PNG/JPG 图片，转为一张页面插入 |
| 插入 PDF 页 | 选择另一个 PDF，选取其中指定页面插入 |

每个有修改的缩略图右上角会出现 **↩** 按钮，点击可单独撤销该页的修改。

### 保存

- **💾 保存**：覆盖原文件（文件名旁的 `●` 标记表示有未保存修改）
- **📤 另存为**：弹出保存对话框，输出到新路径
- **↩ 撤销修改**：一键恢复到文件初始状态

## PDF 拆分（pdf-split）

### 打开文件

- 点击工具栏 **📂 打开文件** 按钮选择 PDF
- 或将 PDF 文件直接**拖拽**到工具区域

### 左侧缩略图选择

缩略图面板支持多种选中方式，用于快速填充自定义范围：

| 操作 | 效果 |
|------|------|
| 单击 | 仅选中该页 |
| Ctrl + 单击 | 追加或取消选中该页 |
| Shift + 单击 | 选中从上次点击到当前页的连续范围 |
| 鼠标拖拽框选 | 框住的缩略图全部选中 |

> 选中后，若多个已选页码之间存在间隔，系统会自动填满间隔页码，确保选中范围始终连续。

### 拆分模式

#### 平均拆分

在右侧配置区输入 **每 N 页为一份**，系统实时计算分段：

- 总页数可被整除时，每份页数相同
- 不足 N 页的余数单独成最后一份

#### 自定义范围

可自由添加多条范围条目，每条定义一个输出文件：

- **起始页 / 结束页**：1-based 页码，可任意指定，范围间允许重叠
- **文件名**（可选）：留空则自动命名为 `{原文件名}_part_{起始页}-{结束页}.pdf`；填写后若无 `.pdf` 后缀则自动追加

**提取选中范围**：在缩略图中选中页面后，点击右侧绿色的 **提取选中范围** 按钮，将自动创建一条对应起止页的范围条目。

### 输出预览

右侧"输出预览"区域实时显示当前配置下的所有输出文件名与页码范围，方便在执行前确认结果。

### 执行拆分

点击 **✂️ 开始拆分** 按钮：

1. 弹出目录选择对话框，选择输出目录后开始执行
2. 按钮变为 **拆分中… (X/Y)** 显示当前进度
3. 完成后底部出现 Toast 提示 `✅ 完成，已保存 X 个文件`，并提供 **📂 打开目录** 按钮直接在资源管理器中打开输出目录

### 文件命名规则

| 条件 | 文件名 |
|------|--------|
| 未填写自定义名 | `{原文件名（不含.pdf）}_part_{起始页}-{结束页}.pdf` |
| 已填写，无 `.pdf` 后缀 | 自定义名 + `.pdf` |
| 已填写，含 `.pdf` 后缀 | 原样使用 |
